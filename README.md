# DelFarm — Онлайн-сервис доставки фермерских продуктов в г. Гродно

Данный проект представляет собой полноценное Fullstack веб-приложение (маркетплейс), позволяющее жителям Гродно заказывать свежую продукцию напрямую у местных фермеров.

## Технологический стек

- **Backend:** Node.js, Express, Sequelize (ORM).
- **Database:** PostgreSQL.
- **Frontend:** React (Vite), Tailwind CSS v4.
- **Безопасность:** JWT (JSON Web Tokens), Bcryptjs (хеширование паролей).
- **Иконки:** Lucide React.

## Функционал
- **Клиент:** Каталог, фильтрация по фермерам и категориям, корзина, оформление заказа с выбором времени доставки, система отзывов.
- **Фермер:** Личный кабинет, дашборд со статистикой, управление товарами и остатками, обработка заказов.
- **Администратор:** Верификация новых фермеров, модерация отзывов.

## Запуск проекта

### 1. База данных
- Создайте БД в PostgreSQL.
- Настройте файл `.env` в корне проекта (используйте `.env.example` как шаблон).
- Выполните `node setup-db.js` для инициализации таблиц.
- Выполните `node seed.js` для наполнения базы тестовыми данными.

### 2. Бэкенд
```bash
npm install
npm start
```
### 3. Фронтенд
```bash
cd client
npm install
